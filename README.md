# SapiTrello-csoport-11
Remélem ezt később lehet változtatni

# Commands
-> Admin
	- ezMode

-> Main
	- new {boardName}
		Create new board.
	- list
		List all boards.
	- remove {boardName}
		Remove board.
	- select {boardName}
		Select board to work in.

-> Board
	- new {cardName}
		Creates new card with name cardName.
	- list
		List all cards.
	- modify {index}
		Modify title, then description of card.
	- remove {index}
		Remove card.
